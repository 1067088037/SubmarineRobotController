# SubmarineRobotController

## 基本信息

### 简介

这是一个用于潜艇控制的Android端App，通过经典蓝牙与Arduino单片机连接，并控制一台拥有6个马达的潜艇。

视觉部分使用了OpenCV和TensorFlow，需要改进的地方很多，但是最终因赛程紧急没能完成。

### 团队

团队共4人，其中2人负责机械，1人负责电控，1人负责软件控制
